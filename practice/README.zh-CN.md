# 实践

## Docker Env

一些 Docker Compose 环境准备脚本。

|名称 | 描述 |
| --- | --- |
| `cross-domain` | 跨域演示环境。描述了多种跨域解决方案的具体实践。 |
| `docker-image`| 多种 `Node.js` 服务端框架使用 `dockerfile` 创建镜像的示例。|

## Node.js Service

使用多个服务端框架创建多个实践示例。

|名称 | 描述 |
| --- | --- |
| `egg` | 使用 `egg` 作为服务端框架。 |
| `express` | 使用 `express` 作为服务端框架。 |
| `koa` | 使用 `koa` 作为服务端框架。 |
| `nest` | 使用 `nest` 作为服务端框架。 |

## Vue3 Frontend

使用 Vue3 创建的前端实践仓库。
